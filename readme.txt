A plugin to display products with filter options using a shortcode. Filtering is handled via AJAX without jQuery.

== Description ==

The GK Product Filter plugin allows you to display a list of products with filter options for size and color. You can use a shortcode to render the product list on any page. Filtering is handled via AJAX using vanilla JavaScript, ensuring a fast and seamless user experience.

== Installation ==

1. Upload the `gk-product-filter` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the `[gk_product_filter]` shortcode on any page to display the product list with filters.

== Frequently Asked Questions ==

= How do I use the plugin? =

After activating the plugin, use the `[gk_product_filter]` shortcode on any page to display the product list with size and color filters.

= Can I customize the appearance of the product list? =

Yes, the plugin uses Bootstrap for styling. You can customize the appearance by modifying the included Bootstrap classes or adding your own custom CSS.
